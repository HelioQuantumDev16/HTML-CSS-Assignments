# CSS Selectors Task

## Description

This Projects demonstrates the use basic HTML and CSS selectors.

## Features

- Element Selector
- Class Selector
- ID Selector
- Child Selectors > First Child > Last Child > nth Child
- Orderlist and unordered list
- Heading

## Files Included

- index.html
- style.css
- README.md

## Technology Used

- HTML5
- CSS

## How to Run

1. Download the Aditya_Task3 project.
2. Open the index.html file in any web Browser.
3. Ensure style.css is in same folder.
4. Output of CSS Selector Task will be displayed.
