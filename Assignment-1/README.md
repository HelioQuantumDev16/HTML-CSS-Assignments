# Laundry Wallah / Laundry mart

## Description

This is a simple HTML project that tell us about Laundry wallah services and their price list.

## Features

- Welcome heading
- Introduction Paragraph
- Laundry image
- Services List
- Price List displayed using table
- Booking Form
- Contact Information

## Files Included

- index.html
- README.md
- laundry.jpg

## How to Run

1. Download the Aditya_Task2 project.
2. Open the index.html file in any web browser.
3. Project Laundry Wallah will be displayed.
